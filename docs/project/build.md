## Build
Before you build be sure to replace the logo located `static/logo.png`

```bash
# generate icons for the application
# This will generate all the icons in `static/icons`
$ npm run make-icons

# for web build
$ npm run build:web

# for all operating platforms
$ npm run build

# for linux platform
$ npm run build:linux

# for windows platform
$ npm run build:win32

# for mac platform
$ npm run build:mas
```
